## Tarea 1 - Ventas de Videojuegos

### Consideraciones:
- Para instalar las dependencias y poder ejecutar todos los codigos sin problemas, estando dentro de la carpeta ```./Tarea-1``` se debe ejecutar:  
```pip install -r requirements.txt```
- La carpeta ```./Codigos```, contiene una carpeta por cada integrante del grupo y “Grupo” corresponde a la visualización generada con IA generativa. Dentro de cada carpeta personal y grupal, se encuentra un archivo de extensión “.py” con la visualización realizada y la grafica generada.
- La carpeta ```./data```, contiene la carpeta ```./raw``` con la fuente de datos intacta y la carpeta ```./clean``` que cuenta con el dataset limpiado, utilizando un código también presente en esa carpeta para limpiarlo.
